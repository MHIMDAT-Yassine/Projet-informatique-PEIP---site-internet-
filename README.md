# Projet Web PEIP 2 - Site HTML/CSS

Ce dépôt contient le code source de notre site web, réalisé en binôme dans le cadre de notre deuxième année de cycle préparatoire PEIP. L'objectif de ce projet était de concevoir un site web complet en appliquant les bonnes pratiques de développement front-end.

## Auteurs du projet

* **Yassine MHIMDAT** 
* **Yanis LABROUZI**   

## Fonctionnalités et points forts

* **Navigation multi-pages :** Le site est composé de 5 pages navigables entre elles.
* **Séparation des préoccupations :** Le style est entièrement défini dans un fichier CSS externe (aucun style inline dans le HTML).
* **Mise en page moderne :** Utilisation combinée de Flexbox et CSS Grid pour structurer efficacement les différents éléments des pages.
* **Design Responsive :** L'interface s'adapte automatiquement à la taille de l'écran (mobile, tablette, ordinateur).

## Technologies utilisées

* HTML
* CSS
